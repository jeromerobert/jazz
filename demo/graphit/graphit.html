GraphIt - A Jazz Applet

University of Maryland
Human-Computer Interaction Lab
GraphIt is a simple Jazz graph drawing program.

In this program you can:
  • Create nodes by clicking the left mouse button on the Jazz surface using the circle tool.
  • Select objects by clicking the left mouse button on them with the arrow tool.
  • Create links between nodes by clicking the left mouse button on them using the linking tool.
  • Pan the Jazz surface by dragging the mouse while pressing the left mouse button using the hand tool.
  • Zoom in (using any tool) by dragging the mouse toward the right while pressing the right mouse button.
  • Zoom out (using any tool) by dragging the mouse toward the left while pressing the right mouse button.
  • Group objects by selecting them and choosing Edit->Group from the Menubar.
  • UnGroup objects by selecting them and choosing Edit->Ungroup from the Menubar.
  • Create a new view of the Jazz surface by choosing View->New View.
